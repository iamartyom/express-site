# How to deploy this site on DigitalOcean?

### 1 step

Create Ubuntu droplet.

### 2 step

Install nodejs, npm and nginx.

```
sudo apt-get update
```

nodejs:
```
sudo apt-get install nodejs
sudo apt-get install nodejs-legacy
```

npm:
```
sudo apt-get install npm
```

nginx:
```
sudo apt-get install nginx
```
